

To run the code:
    1. Download and extract the dataset for both problems in the code folder.
        1.1. The dataset for the problem could be downloaded from here https://www.kaggle.com/mlg-ulb/creditcardfraud?select=creditcard.csv

There are four .py files in the Code folder:
    project.py: The main part of the project.
    config.py: To set the parameters to run the code.
    ovs.py: Includes the code for oversampling the dataset of the second problem.
    uns: ovs.py
